Detailed Description

template<class Key, class T, class Hash = std::hash<Key>, class Pred = std::equal_to<Key>, class Alloc = std::allocator<std::pair<const Key, T>>, size_t SLOT_PER_BUCKET = DEFAULT_SLOT_PER_BUCKET>
class cuckoohash_map< Key, T, Hash, Pred, Alloc, SLOT_PER_BUCKET >::reference

Class returned by operator[] which wraps an entry in the hash table. Note that this reference type behaves somewhat differently from an STL map reference. Most importantly, running this operator will not insert a default key-value pair into the map if the given key is not already in the map.
