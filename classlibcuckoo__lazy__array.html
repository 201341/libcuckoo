template<uint8_t OFFSET_BITS, uint8_t SEGMENT_BITS, class T, class Alloc = std::allocator<T>>
void libcuckoo_lazy_array< OFFSET_BITS, SEGMENT_BITS, T, Alloc >::allocate ( size_t  target)
inline

Ensures that the array has enough segments to index target elements, not exceeding the total size. The user must ensure that the array is properly allocated before accessing a certain index. This saves having to check every index operation.

Parameters
targetthe maximum number of elements we should be able to access after calling this function
