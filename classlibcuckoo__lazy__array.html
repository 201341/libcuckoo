Detailed Description

template<uint8_t OFFSET_BITS, uint8_t SEGMENT_BITS, class T, class Alloc = std::allocator<T>>
class libcuckoo_lazy_array< OFFSET_BITS, SEGMENT_BITS, T, Alloc >

A fixed-size array, broken up into segments that are dynamically allocated upon request. It is the user's responsibility to make sure they only access allocated parts of the array.

Template Parameters
OFFSET_BITSthe number of bits of the index used as the offset within a segment
SEGMENT_BITSthe number of bits of the index used as the segment index
Tthe type of stored in the container
Allocthe allocator used to allocate data
