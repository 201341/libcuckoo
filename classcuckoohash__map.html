inline

Inserts the given key-value pair into the table. The key will be immediately constructed as key_type(std::forward<K>(key)). If the insertion succeeds, this constructed key will be moved into the table and the value constructed from the val parameters. If the insertion fails, the constructed key will be destroyed, and the val parameters will remain valid. If there is no room left in the table, it will be automatically expanded, which may throw exceptions.

Parameters

Returns
true if the insertion succeeded, false if there was a duplicate
